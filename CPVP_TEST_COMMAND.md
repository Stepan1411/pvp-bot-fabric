# Тестовая команда Crystal PVP

## Команда

```bash
/pvpbot cpvptest
```

## Что делает

1. Создаёт тестового бота `CpvpTestBot` рядом с игроком
2. Даёт ему 64 обсидиана и 64 кристалла
3. Телепортирует бота на расстояние 2 блока от позиции установки
4. Выбирает обсидиан в руке
5. Поворачивает голову бота на позицию
6. Выполняет `player CpvpTestBot use once` для установки обсидиана
7. Ждёт 0.5 секунды
8. Проверяет что обсидиан установлен
9. Выбирает кристалл в руке
10. Поворачивает голову на верх обсидиана
11. Выполняет `player CpvpTestBot use once` для установки кристалла
12. Ждёт 0.5 секунды
13. Проверяет что кристалл установлен

## Если тест провалился

### Ошибка: "Обсидиан НЕ установлен!"

Возможные причины:
1. Команда `player use once` не работает (проверьте что Carpet mod установлен)
2. Бот слишком далеко от позиции (дистанция > 4.5 блоков)
3. Бот смотрит не в ту сторону
4. Позиция занята другим блоком

### Ошибка: "Кристалл НЕ установлен!"

Возможные причины:
1. Обсидиан не установлен (см. выше)
2. Над обсидианом нет места (блок занят)
3. Команда `player use once` не работает для кристаллов
4. Бот слишком далеко от позиции

## Что проверяет тест

- ✅ Команда `player use once` работает для установки блоков
- ✅ Команда `player use once` работает для установки кристаллов
- ✅ Бот может выбирать предметы в руке
- ✅ Бот может поворачивать голову на позицию
- ✅ Блоки устанавливаются на сервере (не "фейково")

## Использование

Запустите команду перед началом полного теста Crystal PVP чтобы убедиться что базовая механика работает.

Если тест успешен - можно переходить к полному тесту в бою (см. TESTING_CRYSTAL_PVP.md).
